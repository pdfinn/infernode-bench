# calibration/

Phase-0 calibration set (IB-2). Hand-authored 20 items (4 per
category) used to size the full benchmark before bulk authoring begins.

**Status:** empty until IB-2 is executed. The first calibration file
should land at `calibration/v0.yaml` with the format documented in the
INFERNODE-BENCH-EPIC.md (IB-2 acceptance criteria).

Calibration items are NOT part of any public subset. They live here
only as the empirical input to per-category sizing decisions and the
calibration vs. live-bench discrimination analysis.
